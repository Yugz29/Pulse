# Archives

Ce dossier contient des documents historiques, obsolètes ou conservés pour référence.

Les archives ne sont pas autoritaires. Elles ne doivent pas guider de nouveaux patchs sans revalidation par l'index global, la roadmap, les contrats ou une décision active.

Si une archive contredit `docs/README.md`, la roadmap, un contrat ou une décision, l'archive perd.
