# Lab

Ce dossier contient les concepts Lab, expérimentaux ou non-Core.

Ces documents ne décrivent pas des capacités Core stables. Ils ne doivent pas justifier une feature Core sans contrat et décision explicites.

Si une doc Lab contredit Core Reset, les contrats Core ou une décision active, Core Reset / contrats / décisions gagnent.
